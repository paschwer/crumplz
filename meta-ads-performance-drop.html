ROAS
📉

CPA

3 types of performance drop

Not all drops are the same problem.

crumplz distinguishes between sudden crashes, slow declines, and false alarms — so your response is calibrated to the actual severity.

Type 1

Sudden crash

ROAS drops 30–50% in a single monitoring period. Usually caused by an algorithm disruption, a technical issue (pixel misfiring, landing page down), or a sudden competitive CPM spike. Requires immediate investigation.

Type 2

Slow decline

Performance degrades 5–10% per day over 3–7 days. The most dangerous type — easy to explain away each day ("just some variance") while the compounding loss mounts. Usually creative fatigue or audience exhaustion. crumplz catches this with trend analysis across consecutive checks.

Type 3

False positive drop

Channel-level ROAS drops but overall business performance holds. A surge in organic, email, or direct traffic generates revenue that Meta doesn't claim attribution for. MER holds or improves. crumplz's Shopify cross-validation catches this — no alert is sent.

FAQ

Performance drop questions.

How is crumplz different from Meta's automated rules for performance drops?
Meta's automated rules can trigger actions when a threshold is hit — pausing campaigns, adjusting budgets. crumplz alerts you and lets you decide the response. This prevents automation errors: an automated rule can pause a campaign that's temporarily underperforming due to algorithm variance — and kill a campaign that would have recovered. crumplz gives you the information; you make the call.
Will crumplz alert me for every small performance fluctuation?
No. crumplz uses dynamic baselines per ad set that account for normal variance patterns. A single bad-performing check won't trigger an alert. The system looks for consistent directional movement across multiple checks — the kind of sustained decline that represents a real problem, not day-to-day noise.
Can crumplz monitor performance across multiple countries or markets?
Yes. crumplz monitors at the ad set level across all campaigns in your connected ad accounts. If you run separate campaigns for DE, FR, UK markets, each is monitored independently with its own baseline. Alerts are tagged by campaign and ad account so you always know which market is affected.
